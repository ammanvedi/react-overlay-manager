# Responsive
Based on a media query we can move the position that an overlay sits in. We
may want to do this to prevent overlays from overlapping each other for example.

We do this by passing a `ResponsiveRules` object to the `OverlayContextProvider`

```tsx dark="true"
import {OverlayContextProvider, ResponsiveRules, OverlayPosition} from "react-overlay-manager";

export const rr: ResponsiveRules = {
    [OverlayPosition.TOP_RIGHT]: {
        '(max-width: 900px)': {
            position: OverlayPosition.TOP_FULL_WIDTH,
            constraints: null,
        },
    },
    [OverlayPosition.TOP_LEFT]: {
        '(max-width: 900px)': {
            position: OverlayPosition.TOP_FULL_WIDTH,
            constraints: null,
        },
    },
    [OverlayPosition.TOP_CENTER]: {
        '(max-width: 900px)': {
            position: OverlayPosition.TOP_FULL_WIDTH,
            constraints: null,
        },
    },
};

<OverlayContextProvider responsiveRules={rr}>
    ...
</OverlayContextProvider>
```

As you can see for each position we create a map of media query (one that can be evaluated
by [window.matchMedia](https://developer.mozilla.org/en-US/docs/Web/API/Window/matchMedia)) to another position.

Just concentrate on the `position` property for now, as the constraints property will be
discussed in the related documentation.
